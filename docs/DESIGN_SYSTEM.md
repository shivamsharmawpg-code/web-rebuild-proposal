# Design System

## Typography
- Display: 2.1rem / 1.8rem
- Section heading: 1.4rem
- Body: 1rem
- Metadata: 0.8rem uppercase

## Spacing Scale
- xs: 0.375rem
- sm: 0.75rem
- md: 1.25rem
- lg: 2rem
- xl: 3rem

## Color Tokens
- `--bg-canvas`: #f7f5f2
- `--bg-panel`: #ffffff
- `--text-main`: #1f2a37
- `--text-soft`: #5c6876
- `--brand-main`: #5b21b6
- `--brand-warm`: #c2410c
- `--brand-calm`: #0f766e
- `--border-tone`: #e7dfd3

## Components
- Sticky masthead with adaptive menu
- Spotlight hero with action buttons
- Card clusters for key content
- Announcement strip
- Structured form fields
- Footer utility region

## Breakpoints
- Base: < 900px (single-column)
- Wide: >= 900px (multi-column layouts)
